# Doctor Booking Appointment System

## 📌 Overview

The Doctor Booking Appointment System is a web-based application designed to simplify the process of booking medical appointments. It allows patients to schedule appointments with doctors, view available slots, and manage bookings efficiently.

## 🚀 Features

* User authentication (Login/Signup)
* Browse available doctors
* Book and manage appointments
* View appointment history
* Admin panel for managing doctors and schedules
* Responsive UI for better user experience

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js / Express.js (assumed)
* Database: MongoDB (assumed)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/vaibhavtamang12/Doctor_Boking_Appointment.git
```

2. Navigate to the project directory

```bash
cd Doctor_Boking_Appointment
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
npm start
```

5. Open in browser

```
http://localhost:3000
```
